## Features

#### 프로젝트 특징

##### :horse_racing: 공동배달

최소 주문 금액을 충족시켜 로켓 배송, 배달의 민족 등의 서비스를 넓은 폭으로 이용 가능합니다.

##### :package: 공동택배

쇼핑몰 하한 금액을 맞추어 배송비를 절약할 수 있습니다.

##### :family: 공동구매

대량 주문을 통해 원하는 만큼만 분배 받아 절약 가능합니다.
